Dev Wellbeing

Dev Wellbeing is a digital wellbeing desktop application built specifically for developers to understand, track, and improve their daily work habits.

It automatically monitors application usage, coding time, and activity patterns — helping developers stay productive without burning out.

Download for Windows Source Code

Key Features

App Usage Tracking

Automatically tracks applications and coding activity in real time.

Daily & Weekly Timelines

Visual timelines help you understand how your time is spent.

Focus Awareness

Identify deep-focus sessions and distraction patterns.

Offline First

Runs locally on your system with no cloud dependency.

Privacy Friendly

All data stays on your machine — no tracking, no analytics.

Cross-Platform Ready

Built with Tauri, Rust, and modern web technologies.
